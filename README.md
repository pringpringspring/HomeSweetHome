# 🏡HomeSweetHome

### 프로젝트 소개
* 직관적으로 사용하기 편리한 홈 인테리어 쇼핑몰 플랫폼
* 홈 인테리어에 관해 필요한 상품과 정보를 한 사이트 내에서 쉽고 빠르게 찾을 수 있는 서비스 제공
* 커뮤니티 기능으로 사용자간 교류를 할 수 있는 서비스 제공
